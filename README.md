# 🎓 Campus Marketplace  
### Buy • Sell • Donate – Exclusive for Verified Students  

A secure marketplace platform built exclusively for verified college students to buy, sell, or donate textbooks, electronics, lab equipment, furniture, and more.

---

## 🚀 Problem Statement

Students frequently need affordable academic resources and essentials.  
At the same time, seniors often have unused books, calculators, lab tools, and furniture.

However:
- There is no verified student-only platform
- Trust issues exist in public marketplaces
- No structured college-based system for reuse

---

## 💡 Solution

CampusCart is a secure, college-restricted platform where:

- Only verified students can register
- Users can buy, sell, or donate items
- Buyers can filter by category, condition, and price
- In-app chat enables negotiation
- Ratings & reviews build trust

---

## ✨ Key Features

- 🔐 College Email Verification
- 📸 Post Items with Photos & Price
- 🗂 Category-based Browsing
- 🔎 Advanced Filters (Price, Condition)
- 💬 Real-time Chat (Negotiation)
- ⭐ Rating & Review System
- ❤️ Watchlist Feature
- 🎁 Donation Listings Option

## 🏗 System Architecture

                                   ↕  
                              Real-time Chat (Socket.IO)

---

## 🔐 Security & Trust Model

- College email verification (.edu / institutional email)
- JWT-based authentication
- Encrypted passwords using bcrypt
- Rating & review transparency
- Verified student-only access
